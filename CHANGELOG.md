# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-10

### Added
- Multi-diagram support: ER, UML (Class, Use Case, Sequence, Activity, State, Component, Deployment, Package, Composite, Communication, Interaction Overview), Flowchart, DFD, Wireframe, Circuit, and Logic Circuit diagrams
- Dynamic toolbox that updates per diagram type
- Diagram Picker modal with search, category filtering, and close button
- Canvas rendering with zoom, pan, and grid snapping
- Connector/wire system with orthogonal and straight routing
- Properties panel for editing component labels, positions, and fields
- Keyboard shortcuts: Undo (Ctrl+Z), Delete, Arrow key nudging, tool switching
- Export to PNG and JSON
- Import/Load diagrams from JSON
- Context menu (Rename, Duplicate, Delete)
- Dark/Light theme auto-detection
- Template/example diagrams for common use cases
- Copyright watermark in status bar
- Right-click and F12 prevention
